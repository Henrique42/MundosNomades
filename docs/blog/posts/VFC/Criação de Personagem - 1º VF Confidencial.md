# Criação de Personagem - 1º VF Confidencial

Esta postagem existe como uma forma de auxiliar os jogadores na hora de criar os seus personagens para a primeira edição do *VF Confidencial*, uma série de one-shot derivados da campanha [[Vestígios Fantasmas]].

## Criando um Personagem

No geral, a criação de personagens segue as mesmas [[Criação de Personagem (VF)|regras]] que a campanha principal, o que inclui o uso de *D&D 5e* como sistema. As exceções são as seguintes:

1. Os personagens devem começar no nível 5;
2. Os personagens devem ser membros da [[S.A.P.O.]] há algum tempo *(eles podem já se conhecer ou não, fica a critério de vocês)*;
3. Os personagens devem estar relacionados a **água** de alguma forma *(ex.: piratas, marinheiros, sacerdotes de deuses aquáticos, etc.)*;
4. Vocês podem escolher algum item *comum* ou *incomum* qualquer para começar *(itens não oficiais precisam de aprovação)*.
