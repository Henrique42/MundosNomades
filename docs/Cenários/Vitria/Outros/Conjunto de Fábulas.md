O **Conjunto de Fábulas** é uma série de 22 livros criada pelo espírito [[Leto]] com base nos conhecimentos adquiridos por [[Fídias, o Orador]]. Os livros são adornados pelos símbolos dos [arcanos maiores](https://pt.wikipedia.org/wiki/Arcano_maior) e cada um deles é acompanhado de um item mágico menor, derivado de seu conteúdo.

Segundo Leto, os livros contém uma forma secreta de derrotar [[O Observador]], o que seria a herança de Fídias para o mundo *(Sessão: [[VF34 ― Um Eco, Uma Mácula]])*.
